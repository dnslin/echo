# Generate invite codes on the server

Invite codes will be generated by the business service as six-character uppercase alphanumeric codes using the A-Z and 0-9 character set. SQLite will enforce uniqueness for active invite codes, room creation will retry generation on collision, and client input will be normalized by removing spaces and hyphens and uppercasing before lookup. Client-generated, custom, long-lived, or password-like invite codes were rejected for MVP to preserve the temporary-room model and keep joining simple.
