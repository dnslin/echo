# Use SemVer tags for releases

Echo releases will use SemVer-style application versions with `v`-prefixed Git tags. For v0.1, the application version will be `0.1.0`, the Git tag will be `v0.1.0`, the GitHub Release title will be `echo v0.1.0`, installer assets will be named like `echo-0.1.0-windows-x64.<ext>`, and the About dialog will show `echo 0.1.0 / 朋友内测版`. Hotfixes can increment patch versions such as `0.1.1`.
